# SQL Student Management System

This project is built using SQL and demonstrates database design, data analysis, and table relationships.

## Task 1: Student Management Database

* Create Database
* Create Students Table
* Insert Student Records
* Calculate Average Scores
* Find Top Performer
* Count Students by Grade
* Update Records

## Task 2: Course and Enrollment Management

* Create Courses Table
* Create Enrollments Table
* Apply Foreign Key Relationships
* Perform JOIN Operations
* Calculate Average Grades
* Find Top 3 Students
* Count Failed Students

## Technologies Used:

* MySQL
